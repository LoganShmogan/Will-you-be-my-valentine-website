# Will you be my valentine?

This is a fun, interactive web page created to ask someone special a very important question! It's a lighthearted project built with core web technologies.

## About The Project

The website presents a playful "Will you be my valentine?" question with "Yes" and "No" buttons. It's designed to be a cute and engaging way to pop the question, with a surprise for the user based on their choice.

You can see a live version of the project here: **emmaisasillygoose.win**

## Built With

The project is built using fundamental web technologies:
*   **JavaScript** (51.5%)
*   **HTML** (25.0%)
*   **CSS** (23.5%)

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You don't need anything special! Just a modern web browser like Chrome, Firefox, Safari, or Edge.

### Installation

1.  **Clone the repository**
    Open your terminal and run the following command:
    ```sh
    git clone https://github.com/LoganShmogan/Will-you-be-my-valentine-website.git
    ```
2.  **Navigate to the project folder**
    ```sh
    cd Will-you-be-my-valentine-website
    ```
3.  **Open the website**
    Simply open the `index.html` file in your preferred web browser.
